Sorting Algorithms Explained

Bubble Sort

Bubble Sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. The pass through the list is repeated until the list is sorted.

Time Complexity: O(n²) in worst and average cases.

How it works: It works by repeatedly comparing adjacent pairs of elements and swapping them if they are in the wrong order.

Insertion Sort

        

            Insertion Sort builds the final sorted array one item at a time. It works
            similarly to how you sort playing cards in your hands.
        

        
Time Complexity: O(n²) in worst and average cases; O(n) in the best case (nearly sorted data).

        
How it works: It takes each element from the unsorted section and inserts it into its correct position in the sorted section.

Selection Sort

        

            Selection Sort is an algorithm that repeatedly selects the smallest (or largest)
            element from the unsorted portion of the list and swaps it with the first unsorted element.
        

        
Time Complexity: O(n²) for all cases.

        
How it works: It divides the array into a sorted and an unsorted section and continuously selects the minimum element from the unsorted section.
